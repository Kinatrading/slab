# Market Scan App

Десктопний застосунок на Python/PyQt6 для пошуку вигідних лотів між звичайними та "slab" стікерами у Steam Community Market. Скрипт читає локальні файли `stickers_clean.json` та `stickers_slab_clean.json`, підтягує актуальні ціни з маркету Steam та показує їх у графічному інтерфейсі.

## Вимоги

- Python 3.10+ (перевірено на 3.11)
- Системні залежності для PyQt6 (на Linux — пакети на кшталт `qt6-base-dev`)
- Аккаунт Steam з можливістю відкривати сторінки маркету

## Установка

```bash
python -m venv .venv
source .venv/bin/activate  # на Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск

```bash
python market_scan_app.py
```

Після старту програми натисніть кнопку **"Налаштування"** й заповніть поля:

- **Кукіс** — значення `sessionid` та `steamLoginSecure` (через `;`).
- **Проксі** — кожен з нового рядка у форматі `host:port` або `host:port:user:password` (необовʼязково).
- **Затримка** — пауза між HTTP-запитами в секундах.

Налаштування зберігаються у файлі `settings.json` поруч зі скриптом. Дані про знайдені `item_nameid` та останні ціни кешуються в `market_cache.json`.

## Де взяти cookies Steam

1. Авторизуйтеся у Steam через браузер та відкрийте будь-яку сторінку маркету, наприклад https://steamcommunity.com/market/.
2. Відкрийте інструменти розробника (Chrome/Edge: `F12` або `Ctrl+Shift+I`).
3. Перейдіть на вкладку **Application** → **Storage** → **Cookies** → `https://steamcommunity.com`.
4. Знайдіть ключі `sessionid` та `steamLoginSecure`.
5. Складіть рядок у форматі `sessionid=ВАШЕ_ЗНАЧЕННЯ; steamLoginSecure=ВАШЕ_ЗНАЧЕННЯ` і вставте його у поле **"Кукіс"** в налаштуваннях застосунку. (можна втупу весь блок кукіс бахнути і буде працювати)

> **Порада:** якщо Steam періодично викидає помилку 429, додайте кілька HTTP-проксі та збільшіть затримку між запитами.

## Оновлення списків

Файли `stickers_clean.json` і `stickers_slab_clean.json` містять підготовлений перелік предметів. Щоб оновити дані, замініть ці файли на актуальні версії (формат — масив обʼєктів зі стікерами).

## Відомі каталоги

- `market_cache.json` — створюється автоматично, кеш для Steam API.
- `settings.json` — параметри користувача.
- `stickers_*.json` — довідники зі стікерами.

Додаткових бекенд-сервісів не потрібно: застосунок працює повністю локально, звертаючись напряму до Steam Community Market.
